# Speckled Noise Particle Pack Vol. 1

## What This Pack Is

Speckled Noise Particle Pack Vol. 1 is a texture pack for building stylized particles, soft noise bursts, speckled impacts, atmospheric breakup, and UI/VFX accents. It includes original Luminary Labs speckled noise textures in multiple sizes, animated flipbook variants, starter materials, and demo content for quick preview and setup.

## What's Included

- Static speckled noise textures:
  - `SpeckledNoise_256/`
  - `SpeckledNoise_512/`
  - `SpeckledNoise_1024/`
  - `SpeckledNoise_2048/`
- Flipbook textures:
  - `SpeckledNoise_Flipbooks/`
- Starter materials:
  - `SpeckledNoise_Materials/`
- Demo content:
  - `SpeckledNoise_Demo/SpeckledNoise_Demo.unity`
  - `SpeckledNoise_Demo/SpeckledNoise_Demo_Base.prefab`
  - `SpeckledNoise_Demo/SpeckledNoise_DemoPrefabs/`
  - `SpeckledNoise_Demo/SpeckledNoise_DemoPrefabs/SpeckledNoise_Demo_Variant01.prefab` through `SpeckledNoise_Demo_Variant50.prefab`

`SpeckledNoise_Media/` is internal publishing media for Asset Store submission and is not part of the normal customer usage workflow.

## How To Use It

1. Open `SpeckledNoise_Demo/SpeckledNoise_Demo.unity` to preview the pack.
2. Choose a texture resolution tier based on your target quality and performance needs.
3. Choose or duplicate a starter material from `SpeckledNoise_Materials/`.
4. Assign a speckled noise texture to your particle system, material, or effect setup.
5. If using flipbooks, enable **Texture Sheet Animation** and match the frame layout to the selected flipbook texture.
